About

django.video 📹 is a collection of curated videos about the Django Web Framework put together by people from rmotr.com 🐶.

As part of our Web Development with Django course, we give our students good resources from different Django conferences. Django has an amazing community and experts from everywhere give talks about different topics. 🙌

It's also a great resource for learning Django development 🤓. The project is structured in a way that makes it easy to walk through the different phases of the project development.

We know that most of you trying to learn Django are tired of dumb tutorials that fall short. This project can provide a comprehensive guide and demonstration of what a production-ready Django project looks like.

This is an free and open source project, developed with Python 3 and Django. If you're interested checkout the Github project page. It's ready to be deployed on Heroku, we'll build a guide on how to do it soon, stay tunned.

If you have any question, feel free to contact us at questions@rmotr.com 📫 and follow us on twitter 🐦.

Finally, if you'd like to suggest a video to add, please submit an issue using this Github page.
